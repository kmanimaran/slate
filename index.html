    Introduction

    Welcome to the WeLink API! You can use our API to access WeLink API endpoints, which can get information on various cats, kittens, and breeds in our database.

    You can view API examples in the dark area to the right.

    Authentication

    To authorize, use this code:

    # With shell, you can just pass the correct header with each request
curl "https://welink.com/api/v3/endpoint?apiKey=XXXX"

    Make sure to replace XXXX with your API key.

    WeLink uses apiKey parameter to allow access to the API. You can get an API key by contacting us - contact@welink.com.

    WeLink expects the apiKey parameter to be included in all API requests to the server as a parameter:

    apiKey=<XXXX>

    You must replace XXXX with your personal API key.

    This endpoint retrieves all kittens.

    HTTP Request

    GET http://example.com/api/kittens

    Query Parameters

    Parameter Default Description

    available true If set to false, the result will include kittens that have already been adopted.

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files…just save them to the `includes` folder and add them to the top of your `index.md`’s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
